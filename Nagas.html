The Nagas (2022)

Mobile ARPG - Personal Project

Project Description

This mobile action RPG was a solo development project where I handled all aspects of game creation, from programming to asset production. Built with Unity, the game features real-time combat, state-driven AI, and multiplayer functionality using Fish-Networking.

  • Gameplay & Systems: Designed and implemented core gameplay mechanics, UI, and supporting systems.
  • Networking: Developed multiplayer functionality using Server Authoritative (SA) and Client-Side Prediction (CSP) techniques with Fish-Networking.
  • AI & Combat: Programmed player and enemy logic based on the state machine pattern for fluid and responsive combat.
  • Art & Asset Creation: Modeled, textured, and animated 3D characters, weapons, enemies, and environments. Created animations and visual effects to enhance immersion.
  • Art & VFX Integration: Created and integrated game-ready assets, including characters, weapons, environments, animations, and visual effects.

This project served as a hands-on learning experience, allowing me to develop a broad skill set across programming, networking, and game asset creation while refining my expertise in Unity.
